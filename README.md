# プロジェクトの概要
冷たいジェラートを意識してベースカラーを水色にし、爽やかな雰囲気と可愛らしさを表現しました。　　

ジェラート屋のホームページなのでターゲットを女性や子供に設定し興味を持ってもらえるよう意識し作成しました。
